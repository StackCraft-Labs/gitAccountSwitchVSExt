# Hello World — House Gallery

A Chrome extension that intercepts the search query **`hello world`** on any major search engine and replaces the results page with a beautiful gallery of house images.

**Supported search engines:** Google · Bing · Yahoo · DuckDuckGo

---

## Project Structure

```
gitAccountSwitchExt/
├── manifest.json          # Chrome Extension Manifest V3
├── content.js             # Detects "hello world" in search queries
├── gallery.html           # Gallery page (shown instead of search results)
├── gallery.js             # Fetches + renders images from Pixabay
├── gallery.css            # Responsive dark-themed grid layout
├── generate-icons.html    # One-time tool to generate PNG icons
└── icons/
    ├── icon16.png         # (generated by generate-icons.html)
    ├── icon48.png
    └── icon128.png
```

---

## Setup — 3 Steps

### Step 1 — Get a free Pixabay API key

1. Go to **[https://pixabay.com/accounts/register/](https://pixabay.com/accounts/register/)** (free, no credit card)
2. After registering, go to **[https://pixabay.com/api/docs/](https://pixabay.com/api/docs/)**
3. Copy your **API Key** shown at the top of that page

### Step 2 — Add the API key

Open **`gallery.js`** and replace the placeholder on line 17:

```js
// Before
API_KEY: 'YOUR_PIXABAY_API_KEY_HERE',

// After
API_KEY: '12345678-abcdef1234567890abcdef12',
```

### Step 3 — Generate icons (one time)

1. Open `generate-icons.html` in Chrome (`File > Open File`)
2. Click **Download Icons**
3. Move the 3 downloaded files (`icon16.png`, `icon48.png`, `icon128.png`) into the `icons/` folder

## Loading the Extension in Chrome

```bash
# No build step needed — pure HTML/CSS/JS
```

1. Open Chrome and navigate to:
   ```
   chrome://extensions
   ```
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `gitAccountSwitchExt/` folder
5. The extension is now active ✓

---

## How to Use

1. Open any supported search engine (Google, Bing, Yahoo, DuckDuckGo)
2. Type exactly:
   ```
   hello world
   ```
3. Press **Enter**
4. The search results page is instantly replaced by the house gallery

## Updating / Reloading After Code Changes

After editing any file, reload the extension in Chrome:

1. Go to `chrome://extensions`
2. Find **Hello World — House Gallery**
3. Click the **reload icon** (circular arrow)

## Customisation

All tuneable values live at the top of **`gallery.js`**:

| Key           | Default                       | Description                          |
|---------------|-------------------------------|--------------------------------------|
| `QUERY`       | `'house home architecture'`   | Pixabay search term                  |
| `PER_PAGE`    | `20`                          | Images loaded per page (max 200)     |
| `ORIENTATION` | `'horizontal'`                | `'all'` / `'horizontal'` / `'vertical'` |
| `IMAGE_TYPE`  | `'photo'`                     | `'photo'` / `'illustration'` / `'vector'` |

To change the trigger phrase, edit `content.js` line 10:
```js
const TRIGGER_PHRASE = 'hello world'; // ← change this
```
